Materi: DOM Selection & JS Vanilla

P - Point (Titik Utama)

Materi ini membahas dasar-dasar memanipulasi DOM (Document Object Model) menggunakan JavaScript Vanilla. Fokus utamanya adalah memahami bagaimana HTML, CSS, dan JS bekerja sama untuk menciptakan interaktivitas tanpa bantuan framework luar.

R - Reason (Alasan)

Mengapa ini penting? Memahami JS Native/Vanilla adalah fondasi sebelum masuk ke framework besar (seperti React). Hal ini memungkinkan optimasi kode yang lebih baik, performa yang lebih ringan, dan pemahaman mendalam tentang bagaimana browser memproses input pengguna.

E - Explanation (Penjelasan)

Berdasarkan catatan papan tulis, ada tiga pilar utama yang dipelajari:

  • 1. DOM Selector: Teknik mengambil elemen.
    • getElementById: Mencari elemen berdasarkan ID unik (Sangat Cepat).
    • querySelector: Lebih fleksibel, bisa menggunakan selector Class, ID, atau Tag.
  • 2. Event Listener: Pemicu aksi berdasarkan interaksi pengguna.
    • Contoh: onclick (klik), onchange (perubahan input), dan onpress (tekan tombol).
    • Menggunakan fungsi callback untuk menjalankan logika saat kejadian berlangsung.
  • 3. Data Handling: Mengolah informasi.
    • Mengambil nilai input melalui properti .value.
    • Menampilkan output menggunakan alert() atau console.log() untuk optimasi dan debugging.

P - Point (Kesimpulan)

Dengan menguasai alur Selector → Event → Action, seorang pengembang web memiliki kontrol penuh atas fungsionalitas halaman sebelum akhirnya berpindah ke penggunaan framework modern.
